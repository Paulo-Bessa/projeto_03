O programa que eu fiz abre um menu quando roda que mostra algumas opcoes de escolha. De la podemos chegar a todas as informacoes solicitadas na especificacao do trabalho: dados do brasil, porcentagem a partir de medias moveis, medias moveis dos estados, dados dos estados e informacoes acumuladas. Para fazer o programa eu importei os dados em .csv do ministerio da saude do dia 17.
